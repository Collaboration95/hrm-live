# Support and compatibility

HRM Live supports macOS 12 or later and standard Bluetooth Low Energy Heart
Rate Measurement devices. The app is intended for fitness display and local
CSV recording; it is not a medical device and must not be used for diagnosis,
treatment, or emergency alerts.

Before requesting help, review the README, the release checklist, and existing
issues. Include the app version/commit, macOS version, Python version for a
development run, connection state, and concise reproduction steps. Redact BLE
addresses and do not attach private session data unless a maintainer requests a
sanitized sample through a private channel.

The project does not currently support HealthKit, Apple Watch, cloud sync,
session browsing, Strava/Garmin export, or automatic updates.
